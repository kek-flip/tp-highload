## 1 Тема, аудитория, функционал

[Google Translate](https://translate.google.com/) - онлайн-переводчик компании Google, предназначенный для перевода текстов и веб-страниц.

### Основной функциона

- Перевод текста
- Перевод текста с изображения
- Перевод стайтов
